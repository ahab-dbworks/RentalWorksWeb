Rebuild Solution

	Right-click WebApi project file --> Publish
	Publish to this folder: <repo root>\build\RentalWorksWebApi
	(The first time you do this, you need to setup the folder and define a Profile to use next time. When doing so, click "Advanced Options" and select "Remove Additional Files at Destination")
	Click Publish
